# Paradigmas de Programación 

Repositorio con resoluciones de ejercicios y parciales.

## Contenidos

| Paradigma | Lenguaje | Conceptos Clave |
| :--- | :--- | :--- |
| **Funcional** | Haskell | Inmutabilidad, composición de funciones, Funciones de Orden Superior, Pattern Matching, Evaluación diferida (Lazy Evaluation), Recursividad. |
| **Lógico** | Prolog | Declaratividad, Universo de Discurso, Unificación, Polimorfismo, Recursividad. |
| **Objetos** | Wollok | Polimorfismo, Encapsulamiento, Delegación, Clases, Herencia vs. Composición, Manejo de Excepciones, **Testeo Unitario**|

## Principios Aplicados

* **No repetición de logica:** Hincapié en la no repetición de lógica, abstrayendo código común en funciones reutilizables.
* **Pensamiento Top-Down:** Descomposición en el QUÉ y despues en el CÓMO, enfocandose primero en las tareas principales y complejas.
* **Modularidad:** Uso de funciones auxiliares con responsabilidades únicas para un código mantenible.
